名前

ifcfg - ifconfig による IP 管理を置き換えるシンプルなスクリプト  

書式


ifcfg [ DEVICE ] [ command ] ADDRESS [ PEER ]

 

説明

このマニュアルは ifcfg コマンドを短くドキュメント化している。

これは ifconfig の 1 つのオプション、つまり、IP アドレス管理を置き換える簡単なスクリプトである。 これは、アドレスを追加するだけでなく、必要に応じて、 重複アドレス検出 (Duplicate Address Detection) RFC-DHCP の実行、 インタフェースを共有する他のホストのキャッシュをアップデートする 要求のなかった (unsolicited) ARP の送信、 制御ルートの追加と Router Discovery の再起動などを行う。

 

IFCONFIG - コマンド書式

DEVICE
- エイリアス、サフィックスをコンマで区切って指定できる。

command
- add, delete, stop のいずれか。

ADDRESS
- オプションとしてプレフィックスの長さを付けることをができる。

peer
- ポイントポイントインタフェースのオプションのピアアドレス。

 

注意

このスクリプトは IPv6 で使うのには適していない。
