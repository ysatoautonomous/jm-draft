名前

nstat, rtacct - ネットワーク統計ツール

 

書式

Usage: nstat [ -h?vVzrnasd:t:jp ] [ PATTERN [ PATTERN ] ]
Usage: rtacct [ -h?vVzrnasd:t: ] [ ListOfRealms ]

 

説明

nstat と rtacct はカーネルの snmp カウンターとネットワークインタフェース統計をモニターする 簡単なツールである。

 

オプション

-h, --help ヘルプを表示する。
-V, --version
バージョンを表示する。
-z, --zeros
ゼロのカウンターもダンプする。 デフォルトでは表示されない。
-r, --reset
履歴をリセットする。
-n, --nooutput
何も表示せず、履歴の更新だけを行う。
-a, --ignore
カウンターの絶対値をダンプする。 デフォルトでは、前回使用されたときからの増分が計算される。
-s, --noupdate
履歴を更新しない。 次回の表示では、今回の値を含めた累計が表示される。
-j, --json
JSON フォーマットで表示する。
-p, --pretty
--json と組み合わせて、綺麗に表示する。
-d, --scan <INTERVAL>
デーモンモードで統計を収集する。 <INTERVAL> は計測間隔で、 秒単位である。
-t, --interval <INTERVAL>
平均値を計算する時間間隔。 デフォルトは 60 秒である。
