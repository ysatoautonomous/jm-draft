RESIZEPART

Section: System Administration (8)
Updated: January 2015
Index Return to Main Contents
 

名前

resizepart - パーティションの新しいサイズをカーネルに知らせる。  

書式

resizepart device partition length  

説明

resizepart はパーティションの新しいサイズを Linux カーネルに知らせる。 このコマンドは「パーティションのサイズ変更」の ioctl の簡単なラッパーである。

このコマンドはブロックデバイス上のパーティションを操作しない。

 

引き数

device
ディスクデバイス。
partition
パーティション番号。
length
パーティションの新しい長さ (512 バイトセクタ単位)。

 

関連項目

addpart(8), delpart(8), fdisk(8), parted(8), partprobe(8), partx(8)  

入手方法

resizepart コマンドは util-linux パッケージの一部であり、 https://www.kernel.org/pub/linux/utils/util-linux/ から入手できる。
