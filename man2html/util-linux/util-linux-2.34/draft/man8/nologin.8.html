名前

nologin - ログインを丁重に拒否する。  

書式

nologin [-V] [-h]  

説明

nologin は、アカウントが利用可能でない旨のメッセージを表示して、0 以外で終了する。 これは、シェルのフィールドの代替とすることを意図しており、 アカウントにログインすることを拒否する。

ファイル /etc/nologin.txt が存在する場合、nologin は、 デフォルトのメッセージの代わりに、ファイルの内容を表示する。

nologin の返り値は常に 1 である。

 

オプション

-h, --help
ヘルプを表示して、終了する。
-V, --version
バージョン情報を表示して、終了する。
 

注意

nologin はアカウント毎にログインを無効にする方法である (通常は、http や ftp のようなシステムアカウントに使われる)。 nologin(8) は /etc/nologin.txt をデフォルト以外のメッセージのオプションの情報元として使う。 このファイルの存在有無に関わらず、ログインアクセスは常に拒否される。

pam_nologin(8) は PAM モジュールであり、通常は root 以外の全てのユーザーがシステムにログインするのを防止する。 pam_nologin(8) の機能は /var/run/nologin または /etc/nologin ファイルで制御される。  

著者

Karel Zak  

関連項目

login(1), passwd(5), pam_nologin(8)  

履歴

nologin コマンドは 4.4BSD で登場した。  

入手方法

nologin コマンドは util-linux パッケージの一部であり、 Linux Kernel Archive から入手できる。
