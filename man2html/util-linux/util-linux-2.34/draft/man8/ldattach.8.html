LDATTACH

Section: System Administration (8)
Updated: July 2014
Index Return to Main Contents
 

名前

ldattach - シリアル回線に回線制御 (discipline) を割り当てる。  

書式

ldattach [-1278denoVh] [-i iflag] [-s speed] ldisc device  

説明

ldattach デーモンは、(シリアルデバイスを参照する) 指定した device ファイルをオープンし、データの送受信を処理するために、回線制御 ldisc を付与する。 デーモンはバックグラウンドで、デバイスをオープンし続けて、 回線制御をロードしたままにする。

回線制御 ldisc は名前または数値で指定できる。

回線制御を解除するには ldattach プロセスを kill(1) すること。

引き数がない場合、 ldattach は使用法を表示する。  

回線制御

カーネルリリースに依存するが、以下の回線制御がサポートされている:
TTY(0)
デフォルトの回線制御。 透過的な操作 (raw モード) と、編集機能を持ったよく使われる端末回線 (cooked モード) を提供する。
SLIP(1)
TCP/IP パケットをシリアル回線上で転送する Serial Line IP (SLIP) プロトコルプロセッサ。
MOUSE(2)
RS232 に接続されたポインティングデバイス (シリアルマウス) のデバイスドライバ。
PPP(3)
ネットワークパケットをシリアル回線上で転送する Point to Point Protocol (PPP) プロセッサ。
STRIP(4)
AX25(5)
X25(6)
非同期シリアル回線上で X.25 パケットを転送する回線ドライバ。
6PACK(7)
R3964(9)
Simatic R3964 モジュールのドライバ。
IRDA(11)
Linux IrDa (infrared data transmission: 赤外線データ転送) ドライバ - http://irda.sourceforge.net/ を参照すること。
HDLC(13)
同期 HDLC ドライバ。
SYNC_PPP(14)
同期 PPP ドライバ。
HCI(15)
Bluetooth HCI UART ドライバ。
GIGASET_M101(16)
Siemens Gigaset M101 シリアル DECT アダプタのドライバ。
PPS(18)
シリアル回線 Pulse Per Second (PPS) ソースのドライバ。
GSM0710(21)
GSM 07.10 マルチプレックスプロトコルモデル (CMUX) のドライバ。
 

オプション

-1, --onestopbit
シリアル回線のストップビットを 1 に設定する。
-2, --twostopbits
シリアル回線のストップビットを 2 に設定する。
-7, --sevenbits
シリアル回線のキャラクタサイズを 7 に設定する。
-8, --eightbits
シリアル回線のキャラクタサイズを 8 に設定する。
-d, --debug
ldattach をフォアグラウンドにして、中断とデバッグを可能にする。 また、進捗についての詳細なメッセージを標準エラー出力に出力する。
-e, --evenparity
シリアル回線のパリティを、偶数に設定する。
-i, --iflag [-]value...
シリアル回線の c_iflag ワードの指定されたビットを設定する。 指定される value は数値またはシンボル名である。 value の前にマイナス記号を付けると、指定されたビットがクリアされる。 カンマ区切りの複数の値を指定して、複数のビットを設定または クリアできる。
-n, --noparity
シリアル回線のパリティを、なしに設定する。
-o, --oddparity
シリアル回線のパリティを、奇数に設定する。
-s, --speed value
シリアル回線の速度 (バンドレート) を指定した value に設定する。
-c, --intro-command string
ldattach の呼び出しの前に、シリアル回線に送信される intro コマンドを定義する。 例えば、回線制御 GSM0710 とともに使う場合、モデムを CMUX モードに切り替えるには、 コマンド 'AT+CMUX=0\r' が一般には適切である。
-p, --pause value
ldattach の呼び出しの前に、value 秒スリープする。 デフォルトは 1 秒である。
-V, --version
バージョン情報を表示して、終了する。
-h, --help
ヘルプを表示して、終了する。

入手方法

ldattach コマンドは util-linux パッケージの一部であり、 https://www.kernel.org/pub/linux/utils/util-linux/ から入手できる。
