FINDFS

Section: System Administration (8)
Updated: March 2014
Index Return to Main Contents
 

名前

findfs - ラベルまたは UUID でファイルシステムを見つける。  

書式

findfs NAME=value  

説明

findfs は、システム上のブロックデバイスを検索し、 指定されたタグをもつファイルシステムまたはパーティションを探す。 現在サポートされているタグは以下の通り:
LABEL=<label>
ファイルシステムラベルを指定する。
UUID=<uuid>
ファイルシステム UUID を指定する。
PARTUUID=<uuid>
パーティション UUID を指定する。 このパーティション識別子は、例えば、GUID Partition Table (GPT) パーティションテーブルで サポートされる。
PARTLABEL=<label>
パーティションラベル (名) を指定する。 このパーティションラベルは、例えば、 GUID Partition Table (GPT) または MAC パーティションテーブルでサポートされる。

ファイルシステムまたはパーティションが見つかった場合、デバイス名が 標準出力に表示される。

ファイルシステムとパーティションの完全な概要については、例えば以下で 取得できる。


lsblk --fs

partx --show <disk>

blkid

 

返り値

0
成功
1
ラベルまたは UUID が見つからなかった。
2
使用法のエラー。引き数の数の間違い。不明なオプション。
 

著者

findfs は元々 Theodore Ts'o によって書かれた。 そして、util-linux パッケージのために Karel Zak によって書き直された。  

環境変数

LIBBLKID_DEBUG=all
libblkid デバッグ出力を有効にする。
 

関連項目

blkid(8), lsblk(8), partx(8)  

入手方法

findfs コマンドは util-linux パッケージの一部であり、 Linux Kernel Archive から入手できる。
